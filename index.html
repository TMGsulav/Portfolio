About

A developer learning
by doing

I'm Sulav, a recent SEE graduate from Nepal. I got into coding because I wanted to build things — websites, tools, games. What started with messing around in HTML turned into a full-blown obsession with making software work.

These days I spend my time building web scrapers, crafting front-end interfaces, and learning how to write code that doesn't break (too often). I also make YouTube content about my journey.

I don't have a CS degree. What I have is curiosity, persistence, and a habit of breaking things until I understand how they work.
